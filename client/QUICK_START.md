# Быстрый старт

## Запуск проекта

1. Перейдите в папку client:
```bash
cd client
```

2. Установите зависимости (если еще не установлены):
```bash
npm install
```

3. Запустите dev-сервер:
```bash
npm run dev
```

4. Откройте браузер по адресу, который покажет Vite (обычно http://localhost:5173)

## Что реализовано

✅ **Все экраны из дизайна:**
- Onboarding (4 экрана с переходами)
- Список целей
- Создание цели (3 шага: базовая информация, дедлайн, планирование)
- Интерактивная карта с прогрессом
- Гараж с кастомизацией
- Караваны (список, создание, типы)
- Профиль со статистикой
- Настройки
- Магазин

✅ **Анимации:**
- Плавные переходы между экранами
- Hover-эффекты на кнопках
- Анимация прогресс-баров
- Анимация движения на карте
- Плавное появление элементов

✅ **Интерактивность:**
- Все кнопки кликабельны
- Формы работают
- Навигация между экранами
- Переключение вкладок
- Выбор опций

## Важно

- Все работает **без бэкенда** - данные хранятся в локальном состоянии
- Приложение готово к интеграции с API
- Telegram SDK интегрирован, но работает в fallback режиме для разработки
